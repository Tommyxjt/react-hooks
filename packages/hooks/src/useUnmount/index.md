# useUnmount

在组件卸载时执行清理函数。常用于取消订阅、清除定时器等。

---

## 使用场景

### 设计思路

- `useUnmount` 提供了一种清晰的方式来清理组件的副作用（例如，取消事件监听器、清除定时器等），避免在组件卸载后留下无用的引用。
- 直接返回清理函数，组件卸载时自动调用。

### 使用场景

- 清除组件的副作用：取消订阅、清除定时器、撤销网络请求等。
- 防止组件卸载后仍尝试更新其状态或进行无意义的操作。

## API

```typescript
useUnmount(() => {
  // 清理操作，如清除定时器
});
```

### Params

| 参数     | 说明                     | 类型         | 默认值 |
| -------- | ------------------------ | ------------ | ------ |
| callback | 组件卸载时执行的清理操作 | `() => void` | -      |
